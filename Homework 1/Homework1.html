The HTML Elements I Know

  • Image
    • Official Document
    • Attributes:
      • Alt: This attribute defines an alternative text description of the image.
      • Height: The intrinsic height of the image in pixels.
      • Width: The intrinsic width of the image in pixels.
  • Title
    • Official Document
    • Attributes:
      • None
  • Header
    • Official Document
    • Attributes:
      • None
  • Link
    • Official Document
    • Attributes:
      • href: This attribute specifies the URL of the linked resource. A URL can be absolute or relative.
      • rel: This attribute names a relationship of the linked document to the current document. The attribute must be a space-separated list of the link types values.
      • hreflang: This attribute indicates the language of the linked resource. It is purely advisory. Allowed values are determined by BCP47. Use this attribute only if the href attribute is present.
  • Video
    • Official Document
    • Attributes:
      • Autoplay: A Boolean attribute; if specified, the video automatically begins to play back as soon as it can do so without stopping to finish loading the data.
      • Buffer: An attribute you can read to determine the time ranges of the buffered media. This attribute contains a TimeRanges object.
      • Controls: An attribute you can read to determine the time ranges of the buffered media. This attribute contains a TimeRanges object.
  • Style
    • Official Document
    • Attributes:
      • Type: This attribute defines the styling language as a MIME type (charset should not be specified). This attribute is optional and defaults to text/css if it is not specified â€” there is very little reason to include this in modern web documents.
      • Media: This attribute defines which media the style should be applied to. Its value is a media query, which defaults to all if the attribute is missing.
      • Title: This attribute specifies alternative style sheet sets.

Vocab Definitions

  • HTTP
    • Definition: HTTP is the basic system for transferring files on the internet. It is the base level that makes the internet work in the way that it does.
    • Official Document
  • HTML
    • Definition: HTML is a language of coding that defines the structure of web pages.
    • Official Document
  • Element
    • An element is just a piece of a webpage that contains information.
    • Official Document
  • Tag
    • Definition: A tag is basically the start and end of an element. It creates the element.
    • Official Document
  • Block (CSS)
    • Definition: A block is defined as when you have an element that is underneath, or a line below, another element and above the next element to create a block.
    • Official Document
  • Doctype
    • Definition: Doctype is used to make sure the browser will be able to understand the specific aspects that it needs to.
    • Official Document
  • Head
    • Definition: The head is the part that basically explains the document, such as the author of the description.
    • Official Document
  • Attribute
    • Definition: An attribute further defines a tag, chaning it in a specific way.
    • Official Document
  • Boolean
    • Definition: A boolean is a condition set that can only have the outcomes of true or false.
    • Official Document
  • DOM
    • Definition: The DOM is document model that allows a browser to interact with different nodes of the document.
    • Official Document
  • CSS
    • Definition: CSS is another type of coding language. It defines how a webpage looks in a browser.
    • Official Document
  • Property (CSS)
    • Definition: A property is an aspect that defines how an element to should and be displayed in the browser.
    • Official Document
  • Selector
    • Definition: A selctor describes an element and what rules are applies to them.
    • Official Document
