Good Practice When Writing HTML

  1. Use Side-by-side Windows
  2. Never write more than five lines without testing
    • When writing, it's a good idea to never write more than one element without testing.
  3. When writing an element, always write the opening tag, then the closing tab, then the content between.
  4. Use indentation to tructure your code and make it more human-readable
    • If you are nesting something with an elemnt, follow this process:
      • Write the opening tag
      • Write the closing tag
      • Pkace a blank like between the tag tags (by pressing enter)
      • Make sure the opening tag and closing tag are indented the same amount
      • Now write the content that goes between
      • Open-Close-New Line-Indent
  5. Learn how your editor Syntax highlighting works.
